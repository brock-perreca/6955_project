# Figure outputs

Diagnostic plots written by `src/walker2d/extract_gait_cycle.py` and the
scripts in `src/diagnostics/`. Each script overwrites its own file.

| File | Written by | What it shows |
|---|---|---|
| `gait_cycle_check.png` | `src/walker2d/extract_gait_cycle.py` | Six-panel plot (one per joint) of the extracted single stride, with joint-limit lines. Sanity-check for the cycle picked by the heel-strike detector. |
| `cycle_continuity.png` | `src/diagnostics/diag_cycle.py` | Three looped gait cycles per joint; red lines mark cycle boundaries. Visual check for the seam discontinuity at the wrap-around. |
| `reference_check.png` | (older diagnostic, may be stale) | Reference joint trajectories. Regenerated by an earlier diagnostic before the seam plot was preferred. Kept on disk as historical artifact. |

These are **not part of the build** — they're tools you run by hand
when something feels off about the reference. The PNGs in this directory
are the most recent outputs at the time of the last reorganization
(2026-04-28).
